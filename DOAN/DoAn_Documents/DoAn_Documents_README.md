# HƯỚNG DẪN VÀ TỔNG HỢP TÀI LIỆU CHO EVERSHOP

## 0. Phân tích source code dự án 

Phần này bao gồm các tài liệu phân tích source code dự án được tái sử dụng, bao gồm thông tin về các cấu trúc, tài nguyên, công nghệ, phân tích cấu trúc xây dựng và phương hướng xây dựng.

## 1. Tài liệu giới thiệu nghiệp vụ và phân tích thiết kế hệ thống (Chương 1, Chương 2)

Phần này bao gồm các tài liệu cần thiết cho đồ án, được xây dựng sau khi tiến hành phân tích đánh giá bổ sung

## 2. Tài liệu kiểm thử (Chương 3, Chương 4)

Phần này bao gồm các tài liệu liên quan đến kiểm thử, kế hoạch kiểm thử
